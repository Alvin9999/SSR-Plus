

支持系统：

✅ Debian 系列
   - Debian 9 (stretch)
   - Debian 10 (buster)
   - Debian 11 (bullseye)
   - Debian 12 (bookworm)
   - Debian 13 (trixie)

✅ Ubuntu 系列
   - Ubuntu 18.04 LTS (bionic)
   - Ubuntu 20.04 LTS (focal)
   - Ubuntu 22.04 LTS (jammy)
   - Ubuntu 24.04 LTS (noble)
   - Ubuntu 25.04 (plucky)

⚠️ 其他系统（CentOS / Rocky / Fedora / openSUSE 等）理论上也可用，
   但可能因软件源或依赖问题需要手动调整，请谨慎使用。
